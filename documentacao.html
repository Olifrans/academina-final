🚀 Instalação

O sistema foi desenvolvido para ser portátil e sem dependências.

Passo 1: Download

Baixe o arquivo index.html fornecido.

Passo 2: Execução

Basta dar dois cliques no arquivo. Ele abrirá no seu navegador padrão.

Nota: Para acesso em múltiplos computadores na mesma rede, recomenda-se hospedar o arquivo em um servidor local (ex: XAMPP) ou usar serviços como GitHub Pages.

👥 Gestão de Alunos

O coração do sistema. Aqui você gerencia o ciclo de vida do aluno.

Cadastro

Preencha nome, CPF (validado automaticamente) e plano. O sistema calcula a data de vencimento com base no plano escolhido.

Status do Aluno

  • 🟢 Ativo: Plano em dia. Acesso liberado.
  • 🔴 Inadimplente: Plano vencido. Acesso bloqueado no check-in.
  • ⚫ Bloqueado: Bloqueio manual (ex: comportamento).
  • ⚪ Cancelado: Aluno desligado (Soft Delete).

💰 Controle Financeiro

Controle entradas (mensalidades) e saídas (aluguel, contas) para saber o lucro real.

Dica Pro: Use a função de "Lançamentos" para registrar despesas fixas mensais. O sistema calculará automaticamente seu Lucro Líquido.

🏋️ Prescrição de Treinos

Crie fichas de treino (A, B, C, D) com versionamento.

Se um aluno mudar de treino, o sistema salva a versão anterior como "Histórico" e cria a "v2", permitindo acompanhar a evolução sem perder dados.

❓ FAQ (Perguntas Frequentes)

O sistema funciona sem internet?

Sim! Após carregar a primeira vez, ele funciona 100% offline.

Meus dados estão seguros?

Sim, ficam salvos no seu navegador (LocalStorage). Recomendamos fazer backup semanal (botão de download no topo).

Posso usar em mais de um computador?

Sim, mas os dados são locais. Para sincronizar, você precisaria de uma versão com banco de dados na nuvem ou fazer restore do backup manualmente.
