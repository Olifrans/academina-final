# 💪 GYM PRO ELITE v3.0 - Sistema Profissional de Gestão de Academias

![Version](https://img.shields.io/badge/version-3.0.0-green)
![License](https://img.shields.io/badge/license-Commercial-blue)
![Status](https://img.shields.io/badge/status-Production-ready-brightgreen)

> **GYM PRO ELITE** é a solução definitiva em arquivo único para gestão de academias. Combinei design premium, regras de negócio robustas e uma landing page de vendas integrada em um único arquivo HTML.

---

## 🚀 Funcionalidades Principais

### 👥 Gestão de Alunos
- Cadastro com validação de CPF em tempo real.
- Status automático (Ativo, Inadimplente, Bloqueado, Cancelado).
- Soft delete (histórico preservado).
- Renovação de planos com cálculo automático de vencimento.

### 💰 Financeiro Completo
- Controle de Entradas e Saídas.
- Cálculo de Lucro Líquido em tempo real.
- Relatórios exportáveis (PDF/Excel).
- Configuração de multas e juros.

### 🏋️ Treinos & Check-in
- Prescrição de treinos com versionamento (v1, v2, v3...).
- Check-in com validação de status e limite diário.
- Histórico de frequência e métricas de retenção.

### 🛡️ Segurança & LGPD
- Dados armazenados localmente (localStorage).
- CPF mascarado na interface.
- Backup automático em JSON criptografado.
- Níveis de acesso (Admin, Recepção, Professor).

## 📦 Instalação Rápida

Não requer servidor! Basta seguir estes passos:

1. **Baixe** o arquivo `index.html`.
2. **Abra** em qualquer navegador moderno (Chrome, Edge, Firefox).
3. **Login Padrão**:
   - Usuário: `admin`
   - Senha: `admin`

> **Dica:** Para uso em rede local, hospede em um servidor simples (ex: XAMPP, Python HTTP Server) ou use GitHub Pages.

## 📊 Arquitetura Técnica

- **Frontend:** HTML5, CSS3 (Variáveis CSS), JavaScript (ES6+).
- **Gráficos:** Chart.js.
- **UI/UX:** SweetAlert2, FontAwesome.
- **Exportação:** jsPDF, html2canvas.
- **Persistência:** LocalStorage (Browser).
